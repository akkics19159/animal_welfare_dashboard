# Previous Ontology vs New Reasoning Engine

| Aspect | Previous ontology | New reasoning engine |
| --- | --- | --- |
| Knowledge source | Hard-coded rule list | Configuration-driven knowledge base |
| Context awareness | Minimal | Explicit context evaluation |
| Rule support | Simple affirmative checks | AND/OR/NOT/threshold/weighted/nested rules |
| Contradictions | None | Constraint validation with review flags |
| Confidence handling | Basic regularization | Fusion confidence, quality, uncertainty, and reliability propagation |
| Explainability | Limited text strings | Structured evidence, triggered rules, contradictions, and review recommendations |
| Extensibility | Limited | Plugin-ready interface for future OWL, probabilistic, graph, or LLM integration |
