# Species Taxonomy

Taxonomy is configuration-driven in vision_intelligence/species_taxonomy.json.

Supported baseline species:

- Human
- Dog
- Cat
- Horse
- Cow
- Buffalo
- Goat
- Sheep
- Pig
- Rabbit
- Bird
- Chicken
- Duck
- Monkey
- Elephant
- Bear
- Camel
- Deer
- Unknown Animal
- Unknown Human

## Add New Species

1. Add a new entry in species_taxonomy.json with:
- name
- group
- prompts
2. Restart service to reload taxonomy.

No code changes are required for taxonomy expansion.
