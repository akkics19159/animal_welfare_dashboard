# Knowledge Model

## Species Profiles

The engine currently supports configuration-driven profiles for:

- Human
- Dog
- Cat
- Cow
- Horse
- Goat
- Sheep
- Bird

Each profile contains:

- normal behaviour
- typical physiology
- environmental preferences
- distress indicators
- expected vocalizations

## Behaviour Library

The behaviour ontology includes:

- standing
- walking
- running
- sleeping
- eating
- drinking
- playing
- social interaction
- aggression
- limping
- pacing
- isolation

Each behaviour entry includes normality, risk level, and context dependencies.

## Rule and Constraint Model

Rules are stored as configuration and support:

- AND
- OR
- NOT
- Thresholds
- Weighted evidence
- Nested rules

Constraints are used to detect contradictions and trigger human review.
