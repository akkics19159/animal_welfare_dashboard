# Analytics Guide

Analytics now includes behaviour-focused sections without breaking existing charts:

- Behaviour frequency
- Behaviour transitions
- Behaviour stability
- Risk timeline
- Trajectory/occupancy heatmap proxy

## Data Sources

Primary source is backend history analytics. If backend is unavailable, local history CSV is used.

## Filters

- Date range
- Species
- Behaviour
- Occupancy (best effort)

## Notes

Charts are best effort and adapt to whichever compatible columns are present.
